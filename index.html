GitHub Pages 部署指南

部署步骤指南

1

创建GitHub账号

访问 github.com 注册账号

2

创建新仓库

点击右上角 "+" → "New repository"

Repository name: 你的用户名.github.io
Public
Initialize this repository with a README
3

上传HTML文件

在仓库页面点击 "Add file" → "Upload files"

将考试系统HTML文件拖放至上传区域

建议命名为: index.html
4

启用GitHub Pages

进入仓库 Settings → Pages

Branch: main (或master) → /(root)

点击 Save

5

访问您的网站

等待1-2分钟后访问：

https://你的用户名.github.io

现在可以分享这个链接给任何人了！
